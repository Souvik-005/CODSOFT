# 🔐 Password Generator (Python)

A simple Python program that generates secure passwords based on user-selected complexity.

## 🚀 Features

* Generate random passwords
* Three password complexity levels:

  * **Low** – Letters and digits
  * **Moderate** – At least one uppercase letter and one digit
  * **Strong** – Includes uppercase, lowercase, digits, and symbols
* User-defined password length
* Input validation using exception handling
* Randomized password order for better security

## 🛠 Technologies Used

* Python
* Built-in Python modules:

  * `random`
  * `string`
